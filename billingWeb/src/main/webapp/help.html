Pahana Edu System Usage Guidelines
Welcome to Pahana Edu

The Pahana Edu system is a web-based platform designed to manage administrative tasks such as billing, inventory, customers, and reports. This guide provides step-by-step instructions for new users to navigate and use the system effectively.

Accessing the System
  • Login: Use your credentials to log in via the login page.
  • Browser: Use a modern browser (e.g., Chrome, Firefox, Edge) for the best experience.
  • Responsive Design: The system is mobile-friendly. On smaller screens, use the hamburger menu () to toggle the sidebar.
Navigating the Interface
  • Sidebar: Access all modules (Dashboard, Billing, etc.) from the left sidebar.
  • Main Content: Displays the selected module in the main frame.
  • Footer: Contains copyright information and links to Privacy Policy and Terms of Service.

Creating a New Bill
  1. Select Customer: Click the Select Customer button, search for a customer, and click Select.
  2. Add Items: Click Add Item, select an item, and enter the quantity (must not exceed available stock).
  3. Review and Save: Verify the total amount and click Save Bill to generate a printable bill.
  4. Clear Form: Click Clear to reset the form.
Viewing Recent Bills
  • Click View Recent Bills to see a list of recent bills.
  • Use the View button to see bill details or the Print button to print a bill.

Filtering Bills
  • Use filters (Customer, Bill Number, Account No, Date Range) to narrow down the list of bills.
  • Click Apply Filters to update the table or Clear Filters to reset.
Viewing and Printing
  • Click the View button to see detailed bill information.
  • Click the Print button to generate a printable bill.

Searching and Filtering Items
  • Search by Name, Description, or Category.
  • Use Stock Range boxes (0-50, 50-100, 100+) to filter items by stock quantity.
  • Click Search to apply filters or Clear to reset.
Creating and Updating Items
  • Click Create New Item to open the create modal and enter item details.
  • Click the Edit button on an item to update its details.
  • Click the Delete button to remove an item (confirmation required).

Searching and Filtering Categories
  • Search by Category Name or Description in the search form.
  • Click Search to filter categories or Clear to reset the form and view all categories.
Creating and Updating Categories
  • Click Create New Category to open the create modal, enter the category name and optional description, and click Create Category.
  • Click the Edit button on a category to update its name or description in the update modal.
  • Click the Delete button to remove a category. Note: Deleting a category will set the category of linked items to null.
Refreshing Data
  • Click the Refresh button to reload all categories and clear any applied filters.

Searching and Filtering Customers
  • Search by Name, Gender, NIC, Email, Phone, Account Number, or Status (Pending, Approved, Rejected).
  • Click Search to filter customers or Clear to reset the form and view all customers.
Creating and Updating Customers
  • Click Create New Account to open the create modal, enter customer details (Name, Gender, DOB, NIC, Address, Email, Phone, Account Number), and click Create Account.
  • Click the Edit button on a customer to update their details in the update modal.
  • Click the Delete button to remove a customer (confirmation required).
Managing Customer Status
  • Click the Approve button () to set a customer’s status to Approved.
  • Click the Reject button () to set a customer’s status to Rejected.
Refreshing Data
  • Click the Refresh button to reload all customers and clear any applied filters.

  • Check Stock: Always verify item stock before adding to a bill.
  • Clear Filters: Reset filters to view all data if results are limited.
  • Customer Status: Ensure customers are Approved before creating bills for them.
  • Category Management: Organize items into categories to improve search efficiency in the Billing module.
  • Print Layout: Ensure your browser's print settings are set to "Fit to Page" for optimal bill printing.
  • API Connectivity: Ensure the backend API (`http://localhost:8080/billingServices/resources`) is running to avoid errors.
  • Toast Notifications: Pay attention to success or error messages for feedback on actions.

If you encounter issues or have questions:

  • Contact the system administrator via email: support@pahanaedu.com.
  • Check the Privacy Policy and Terms of Service for additional information.
  • Review the system logs in the browser console (F12) for error details if needed.
